# Dependencias y Librerías - DentiEstefy

Este documento detalla las librerías utilizadas en el proyecto y su propósito.

## Dependencias de Producción (`dependencies`)

Estas son las librerías esenciales para que la aplicación funcione en el navegador:

| Librería | Versión | Descripción |
| :--- | :--- | :--- |
| **React** | `^19.0.0` | Biblioteca principal para construir la interfaz de usuario basada en componentes. |
| **React DOM** | `^19.0.0` | Paquete que sirve como entrada a los métodos del DOM para React. |
| **React Router DOM** | `^7.1.0` | Gestión de la navegación y rutas dentro de la aplicación. |
| **Lucide React** | `^0.469.0` | Set de iconos vectoriales hermosos y consistentes para la interfaz. |

## Dependencias de Desarrollo (`devDependencies`)

Herramientas utilizadas durante el desarrollo para compilar, tipar y optimizar el código:

| Librería | Versión | Descripción |
| :--- | :--- | :--- |
| **Vite** | `^6.0.0` | Herramienta de compilación (bundler) extremadamente rápida para desarrollo moderno. |
| **@vitejs/plugin-react** | `^4.3.0` | Plugin oficial de Vite para dar soporte a React (HMR, Fast Refresh). |
| **@types/react** | `^19.0.0` | Definiciones de tipos de TypeScript para React. |
| **@types/react-dom** | `^19.0.0` | Definiciones de tipos de TypeScript para React DOM. |

## Estructura de Estilos
El proyecto utiliza **CSS Vanilla** (CSS estándar) organizado por componentes y páginas, lo que permite un control total sobre el diseño sin dependencias externas de frameworks de CSS complejos.
